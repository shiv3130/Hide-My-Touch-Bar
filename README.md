# Touch Bar Free

Touch Bar Free is an original local macOS menu-bar utility that presents a blank Touch Bar overlay when you double-tap your chosen key. Double-tap again to restore the normal Touch Bar.

It is free for personal use and has no license key, activation, trial, subscription, or purchase check.

This project does not include copied code or assets from Hide My Bar. The public Hide My Bar Community repository contains product metadata/documentation, not reusable app source under an open-source license. This app uses Apple private Touch Bar APIs, so it is intended for local installation on your Mac and is not App Store-safe.

The built app is:

`Touch Bar Free.app`

Open the `TB` menu bar item and choose `Preferences...` to select the double-tap key. Supported keys are Command, Control, Option, Shift, Fn / Globe, Escape, and F1-F12.

If the selected key does not toggle the Touch Bar, allow Touch Bar Free in System Settings -> Privacy & Security -> Input Monitoring, then quit and reopen the app.
